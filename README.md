🚀 Features

👤 User Side:
Browse property listings

View detailed property information

Search & filter properties

View property location on map

Submit inquiries about properties

Contact system

Request construction services

View upcoming projects

🛠️ Admin Side:

Secure admin authentication

Add / edit / delete properties (CRUD)

Manage upcoming projects

View user inquiries & messages

Handle construction service requests

Full control over platform content

🧠 System Architecture:

Frontend communicates with backend via REST APIs

Backend handles authentication, CRUD operations, and requests

PostgreSQL stores all structured data

Admin routes protected via authentication middleware

🛠️ Tech Stack:

Frontend
HTML5
CSS3
JavaScript

Backend
Node.js
Express.js

Database
PostgreSQL (Neon)

Deployment
Frontend: Netlify
Backend: Railway
Database: Neon Serverless PostgreSQL
